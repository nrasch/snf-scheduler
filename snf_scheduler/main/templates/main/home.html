Skilled Nursing Facility (SNF) Prototype Notes:

  • API Endpoints: The current endpoints are listed below. Access requires an API token, which can be generated via the "API Tokens" link in the left navigation menu. The tokens utilize Django-Rest-Knox for authentication.
  • CRUD Functionality via UI: CRUD operations are available leveraging DataTables, AJAX, and Django. You can find links to these areas on the left-hand navigation menu.
  • Browsable API: There is also a browsable API interface. You can find the link to it on the left-hand navigation menu.
  • Access Control: Access to the UI and API are handled by groups defined in the Admin Panel or via Knox token.
    • If you are viewing this page with a staff account will will see a link to the Admin Panel on the left-hand navigation menu.
  • Work in Progress: This is a work in progres. More features, business logic, and documentation coming soon...

Current API Endpoints

API Root

GET /

Root endpoint for the API. Provides links to other resources.

SNFs

GET, POST /snfs/

List or create SNFs.

GET, PUT, PATCH, DELETE /snfs/{id}/

Retrieve, update or delete a specific SNF by ID.

Patients

GET, POST /patients/

List or create patients.

GET, PUT, PATCH, DELETE /patients/{id}/

Retrieve, update or delete a specific patient by ID.

Appointments

GET, POST /appointments/

List or create appointments.

GET, PUT, PATCH, DELETE /appointments/{id}/

Retrieve, update or delete a specific appointment by ID.

Appointment Notes

GET, POST /appointment-notes/

List or create appointment notes.

GET, PUT, PATCH, DELETE /appointment-notes/{id}/

Retrieve, update or delete a specific appointment note by ID.

Patient Notes

GET, POST /patient-notes/

List or create patient notes.

GET, PUT, PATCH, DELETE /patient-notes/{id}/

Retrieve, update or delete a specific patient note by ID.
